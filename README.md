# Robotics Automated Funding

## Description
This is a program to help automate the process of sending emails to possible future sponsors for robotics. It will take a given message and send it to all given emails.

## Windows Installation Guide

1. Install python at https://www.python.org/ if you haven't already
2. Open command prompt and enter the command py --version if this works than you have successfully installed python
3. Go to the releases page on the github repo: https://github.com/JonathanElazar/Robotics-Automated-Funding/releases and download the zip folder
4. Open the config folder put your credentials for your email account in credentials.py put the message you want to send in message.txt and the sponsors you would like to send messages to under possible-sponsors.txt
5. Navigate back to the folder with main.py and run the command py main.py
